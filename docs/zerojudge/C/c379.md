# c379. 成為出題者

## Java
```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        int a = scanner.nextInt();

        double b = a * 0.3;

        System.out.println((int) b);
        
        scanner.close();
    }
}

```

## C++
```cpp
#include<iostream>
using namespace std;
int main() {
    int number;
    cin >> number;
    cout << number * 0.3;

    return 0;
}
```
