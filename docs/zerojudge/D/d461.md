# d461. 班際籃球賽

## Java
```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        int number = scanner.nextInt();
        System.out.println(number - 1);

        scanner.close();
    }
}
```

## C++
```cpp

}
```
