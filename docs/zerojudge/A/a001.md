# a001. 哈囉

## Java
```java
import java.util.Scanner;

public class Main {
    public static void main (String[]args) {
        Scanner scanner = new Scanner(System.in);
        String word = scanner.nextLine();
        System.out.println("hello, " + word);
        scanner.close();
    }
}
```

## C++
