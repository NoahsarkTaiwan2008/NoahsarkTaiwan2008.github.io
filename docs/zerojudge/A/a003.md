# a003. 兩光法師占卜術

## Java
```java
import java.util.Scanner;

public class Main {
    public static void main (String[]args) {
        Scanner scanner = new Scanner(System.in);
        int m = scanner.nextInt();
        int d = scanner.nextInt();
        int s = (m * 2 + d) % 3;

        if (s == 0) {
            System.out.println("普通");
        } else if (s == 1) {
            System.out.println("吉");
        } else if (s == 2) {
            System.out.println("大吉");
        }

        scanner.close();
    }
}
```

## C++
```cpp
#include<iostream>
using namespace std;
int main() {
    int m, d, s;
    cin >> m >> d;
    s = (m * 2 + d) % 3;
    
    if (s == 0) {
        cout << "普通";
    } else if (s == 1) {
        cout << "吉";
    } else if (s == 2) {
        cout << "大吉";
    }

    return 0;
}
```
