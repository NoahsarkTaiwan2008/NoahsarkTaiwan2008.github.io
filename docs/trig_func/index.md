# 三角函數

$$\tan\left(-\frac{4\pi}{3}  \right)$$

$$\Rightarrow \frac{-4\pi}{3}×\frac{180}{\pi}=-240$$

$$\Rightarrow -240 + 360 = 120$$

120位於第二象限

$$\Rightarrow \frac{-\sqrt{3}}{1}=-\sqrt{3}$$

$$Ans = -\sqrt{3}$$

---

$$\sin\frac{3\pi}{2}$$

$$\Rightarrow \frac{3\pi}{3}×\frac{180}{\pi}=270$$

$$\Rightarrow 360-270=90$$

90位於第一象限

$$Ans=\frac{1}{\sqrt{2}}$$
