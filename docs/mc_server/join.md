# 伺服器連線

要進行伺服器連線，首先需要確認**伺服器**和**你的電腦**所連接的網路是否**相同**。
要檢查網路IP，**Windows**請開啟**命令提示字元**或**PowerShell**，MacOS請開啟**terminal(終端機)**，並輸入`ipconfig`，尋找`IPv4`的選項，確認後方的網路IP是否相同。

## 伺服器與遊玩的電腦連接同個網路

開啟Minecraft。
選擇**多人遊戲**之後，在伺服器IP的位置輸入`localhost`或是剛才輸入`ipconfig`之後查看的網路IP後即可連線。

## 伺服器與遊玩的電腦連接不同網路

如果伺服器與遊玩的電腦連接不同網路，那麼有以下幾種不同的連線方式:
- 進行**port forwarding**(比較麻煩，需要進行路由器設定)
- 使用VPN連線(使用**Radmin VPN**進行連線)
